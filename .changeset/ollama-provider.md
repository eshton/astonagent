Add an `ollama` provider for Ollama Cloud and local daemons. Ollama exposes an OpenAI-compatible endpoint, so it reuses the OpenAI adapter with Ollama's base URL (`https://ollama.com/v1` by default) and `OLLAMA_API_KEY`. Import via `@astonagent/providers/ollama`.
